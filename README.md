## 기능 목록

- 가장 인기있는 영화
- 영화 검색
- 영화 상세 모달
- 영화 상세 모달 내 별점 매기기
- 영화 리스트 무한 스크롤 더 가져오기

## 실행 방법

1. 레포지토리 클론

```bash
git clone -b leejin-yang --single-branch https://github.com/woowacourse/javascript-movie-review.git
```

2. 의존성 설치

```bash
npm install
```

3. 실행

```bash
npm run start
```
